# bashbud - Boilerplate and template maker for bash scripts 

## usage

`bashbud` can be used to quickly create new scripts with
cli-option support and automatic documentation applied.

```text
--new|-n   [GENERATOR] **TARGET_DIR**
--bump|-b  [PROJECT_DIR]
--help|-h
--version|-v
```


`--new`|`-n`  

`--bump`|`-b`  

`--help`|`-h`  

`--version`|`-v`
